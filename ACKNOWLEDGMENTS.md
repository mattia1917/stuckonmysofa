# Acknowledgments

## Data Sources

The authors acknowledge the following platforms and datasets used to build the project:

- **MyAnimeList** (myanimelist.net) — manga titles, genres, themes, demographics and scores. Dataset sourced via Kaggle: `duongtruongbinh/manga-and-anime-dataset`.
- **Kaggle** (kaggle.com) — two comic book datasets:
  - `deepcontractor/marvel-comic-books` — Marvel Comics series and issues
  - `deepcontractor/dc-comic-books-dataset` — Complete DC Comic Books collection
- **Goodreads** (goodreads.com) — ratings and metadata reference for graphic novels and western comics.

## Libraries and Tools

This project was built using the following open-source tools:

- **scikit-learn** — TF-IDF vectorization (`TfidfVectorizer`) and cosine similarity (`cosine_similarity`)
- **pandas** — dataset loading, filtering and manipulation
- **numpy** — numerical operations
- **Python 3.10** — programming language
- **Anaconda** — environment and package management
- **Visual Studio Code** — development environment
- **GitHub** — version control and project hosting

## AI Tools

Generative AI platforms (Claude by Anthropic, ChatGPT by OpenAI) were used to assist with code structuring and dataset processing.

## Authors' Contributions

The project was developed by a three-person team for the Data Mining and Text Analytics course (A.A. 2025-2026), supervised by Dr. Alessandro Bruno at IULM University.

**M.** led the overall project design and concept (StuckOnMySofa), coordinated the team's work, and conducted research on content-based recommendation system techniques. M. also contributed to dataset assembly and curation.

**J.** handled the technical implementation of the TF-IDF vectorization pipeline and cosine similarity algorithm. J. managed the conda environment configuration and ensured the code ran correctly across different machines. 

**A.** curated and assembled the dataset of 15,797 comics and graphic novels from three distinct sources, including genre classification and content tagging. 

All team members collaborated on testing, debugging, and uploading the final version of the project to GitHub.
